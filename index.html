PERSONAL PROJECT

CHAT WITH PDF

An AI-powered web application that allows users to upload PDF documents and ask questions in natural language.

Technology stack: The platform uses a modern technology stack for performance, scalability, and security. It uses Next.js for server-side rendering, React for a user-friendly interface, TypeScript for code maintainability, Tailwind CSS for responsive design, and Clerk for secure user authentication. The back-end uses Drizzle ORM, PostgreSQL, AWS SDK, OpenAI API, OpenAI Edge, Neo Database Serverless, and Pinecone vector database. Additional libraries like Drizzle-kit, @tanstack/react-query, @clerk/nextjs, clsx, tailwind-merge, and Axios facilitate HTTP communication with APIs.
